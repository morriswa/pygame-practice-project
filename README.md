# Jenna's Dream Video Game :)

## Env setup (macos)

- navigate to root project dir
-     brew install python@3.11
-     python3.11 -m venv .
-     source bin/activate
-     which python
- verify output to begin with 
project dir and end with bin/python
-     pip install pygame
- ensure demo app works by running:
-     python ./src/JennaGameRunner.py
